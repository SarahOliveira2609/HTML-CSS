Principais formatações

Negrito/Destaque

Nesta paragráfo estamos usando um termo em negrito usando a tag B (não semântica)

Nesta frase, temos um termo em destaque usando a tag STRONG (semântica)

Itálico/Ênfase

Nesse paragráfo estamos usando um termo em itálico usando a tag I (não semântica)

Nest afrase, temo um termo em ênfase usando a tag EM(semânica)

Texto marcado

Podemos criar também um texto marcado usando a tag MARK

E no outro parágrafo temos outro texto marcado no final

Texto grande e pequeno

Estamos criand um texto grande e um texto pequeno neste parágrafo

Tesxto deletado

Podemos marcar um texto excluido para indicar que ele deve ser lido, mas não considerado

Texto inserido

Podemos marcar um texto como inserido para dar um ênfase e indicar que ele foi adicionado depois

Existe também o sublinhado com a tag U (não semânica)

Texto sobrescrito

Para inserir coisas tipo x20+3 ex: x20+3

Texto subscrito

Para inserir coisas do tipo H2O ex: H2O
